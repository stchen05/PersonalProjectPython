# PersonalProjectPython

A minimal Python project scaffold created by an assistant.

Getting started

- Create a virtual environment (optional):
  python -m venv .venv
- Activate (PowerShell):
  .\.venv\Scripts\Activate.ps1
- Install dev requirements:
  pip install -r requirements-dev.txt
- Run tests:
  python -m pytest -q

Project layout

- src/personal_project: package code
- tests: pytest tests
